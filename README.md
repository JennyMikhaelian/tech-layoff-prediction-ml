# Predicting Corporate Layoffs

This project analyzes corporate layoffs and predicts company status (private or public) using machine learning.

## Authors
Jennifer Mikaelian  
Elias Bou Moussa  
Edward Habra  

## Project Description
Tech layoffs have become increasingly common due to economic pressure, over-hiring, and strategic changes in companies.  
This project analyzes a dataset of corporate layoffs and builds a machine learning model to classify company status.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Machine Learning Model
We used the **Random Forest Classifier** to predict company status.

## Model Performance
Accuracy: **~80%**

Evaluation metrics used:
- Confusion Matrix
- Accuracy Score
- Classification Report
- ROC Curve
